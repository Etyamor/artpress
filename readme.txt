== Description ==

ArtPress is a lightweight WordPress theme designed for blogs and creative content. It features a clean layout with a focus on readability and performance.

= Features =

* Fully responsive design for mobile, tablet, and desktop
* Responsive header with mobile navigation menu
* Blog listing with sidebar layout
* Single post template with featured images
* Newsletter subscribe section
* Sidebar with Categories and Tag Cloud widgets
* Pagination support
* Translation-ready

== Installation ==

1. In your WordPress admin, go to "Appearance > Themes > Add New".
2. Click "Upload Theme" and select the `artpress.zip` file.
3. Click "Install Now", then activate the theme.

Alternatively, extract the `artpress` folder into `/wp-content/themes/` and activate it from "Appearance > Themes".

== Frequently Asked Questions ==

= Does this theme support menus? =

Yes. The theme supports 4 menu locations:

* Primary Menu — main header navigation
* Footer Menu — footer links
* Footer Categories — footer category links
* Footer Legal — footer legal/policy links

Go to "Appearance > Menus" to create and assign menus to these locations.

= Does this theme support widgets? =

Yes. The theme includes a sidebar widget area. Go to "Appearance > Widgets" to add widgets such as categories or tag cloud.

= How do I add a newsletter subscribe section? =

The newsletter subscribe section is included in the footer by default. Submissions are handled by the theme and delivered to the site owner's admin email.
